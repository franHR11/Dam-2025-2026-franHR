# Proyecto DAM 2025-2026 FranHR

Este proyecto contiene ejercicios y prácticas de Desarrollo de Aplicaciones Multiplataforma.

## Descripción
Un conjunto de ejercicios prácticos para aprender desarrollo de interfaces, acceso a datos, etc.

## Características
- Ejercicios de desarrollo de interfaces
- Prácticas de programación
- Aplicaciones web simples
- Análisis de motores de juegos con efectos 3D y parallax
- Sistema de gestión de clientes usando interfaces XML y Flask con SQLite
- Instalación y configuración de Odoo con módulo personalizado para gestión de notas

## Instalación
Clona el repositorio y abre en tu editor favorito.
