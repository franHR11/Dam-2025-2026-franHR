# Proyecto DAM 2025-2026 - FranHR

## Descripción Comercial
Este proyecto recopila ejercicios y prácticas del ciclo de Desarrollo de Aplicaciones Multiplataforma (DAM), enfocándome en aprender conceptos clave como acceso a datos, desarrollo de interfaces, programación de servicios y multimedia. Todo desarrollado con código minimalista y funcional.

## Características Principales
- **Acceso a Datos:** Manejo de ficheros, conectores, ORM y BD objeto-relacionales.
- **Desarrollo de Interfaces:** Generación de UI, componentes visuales, diseño de interfaces, aplicaciones gráficas AWT.
- **Programación de Servicios:** Multiproceso, multihilo, comunicaciones en red, servicios web. Incluye generación de consultas SQL usando servicios locales como Ollama para gestión de datos en apps de pesca y caza.
- **Multimedia y Dispositivos:** Motores de juegos, desarrollo 2D/3D, librerías multimedia, apps móviles. Incluye ejercicios de clases en Python para simular pesca de peces y personajes en juegos isométricos con HTML y JavaScript. También prácticas con Three.js para escenas 3D, cubos giratorios y raycasting para interacciones con mouse.
- **Sistemas Empresariales:** ERP-CRM, instalación y configuración.

## Instrucciones de Instalación y Uso
1. Clona el repositorio.
2. Para ejercicios Java, compila con `javac *.java` y ejecuta con `java Principal`.
3. Para ejercicios Python, ejecuta con `python archivo.py`.
4. Para otros lenguajes, sigue las instrucciones específicas en cada carpeta.
5. Asegúrate de tener Java y Python instalados y configurados en PATH.

## Tecnologías Utilizadas
- Lenguajes: Java, Python, posiblemente otros según ejercicios.
- Frameworks: Básicos de Java y Python, sin librerías externas innecesarias.
- Bases de Datos: MySQL u otras según especificado.
